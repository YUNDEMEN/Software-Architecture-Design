* 概念架构设计要领 1： 功能需求与质量需求并重
  * 如何从功能需求向设计过渡
  * 如何从质量需求向设计过渡
    
    解决该问题就要用到“场景技术”，其关键是**使笼统的非功能目标明确化**，
    
    **包含5要素：**
    
    1. 影响来源
    
    2. 如何影响
    
    3. 受影响对象
    
    4. 问题
    
    5. 所处环境
    
    **场景思维的工具**
    
    1. 场景卡： 是“关键点”（用于识别场景）
    
    2. 目标--场景--决策表： “纵贯线”（用于打通思维）
